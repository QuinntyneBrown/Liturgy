## L1-UX-001: Design System & Visual Language
*(formerly L1-012)*

The system must implement a single, documented design system — FaithTech's color tokens, phase
color coding, three-typeface hierarchy, and a fixed set of BEM components — and must render the
signature elements (the Rhythm Rail, the 5R dial, pip strips, gates, work cards) consistently
across every screen.

## L1-UX-002: Responsive & Accessible UI
*(formerly L1-013)*

The system's web UI must be usable across mobile, tablet, and desktop viewports and must meet
baseline accessibility: keyboard operability, skip-to-content, meaningful ARIA state on
interactive enforcement controls, visible focus, and reduced-motion support.
